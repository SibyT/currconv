# Project Title

Currency Converter. 
Converts currency based on todays rate. It uses [http://fixer.io/] rest api for currency rate. 

## Getting Started

### Prerequisites


```
Java 1.8
Apache Maven 3.5.2
```
### Installing
````
git clone.

cd currconv
mvn clean install
mvn package
mvn spring-boot:run

````

Open [http://localhost:8080/]
 for other ports run
 ````
mvn -Dserver.port=9080 spring-boot:run
````
Open [http://localhost:9080/]
